## 🚀 Getting Started

### Prerequisites

To run this portfolio locally, you only need:

- 🌐 **Modern Web Browser** (Chrome, Firefox, Safari, Edge)
- 📝 **Code Editor** (VS Code, Sublime Text, Atom) - for customization

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/NahianAhmed/NahianAhmed.github.io.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd NahianAhmed.github.io
   ```

3. **Open in browser:**
   ```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```

**That's it!** No build process, no dependencies to install. Just open and enjoy! 🎉

## 📱 Usage

### 👀 Viewing the Portfolio
- **Local**: Open `index.html` directly in any modern browser
- **Live**: Visit [nahianahmed.github.io](https://nahianahmed.github.io/)

### ✏️ Customization
- **Content**: Edit `index.html` to update personal information, skills, and experience
- **Styling**: Modify CSS custom properties in the `<style>` section for colors and effects
- **Images**: Replace images in the `assets/` folder (maintain same names or update references)

### 🎨 Color Customization
Update CSS custom properties at the top of the `<style>` section:
```css
:root {
    --primary-color: #6366f1;    /* Main accent color */
    --secondary-color: #8b5cf6;  /* Secondary accent */
    --accent-color: #06b6d4;     /* Highlight color */
    /* ... more variables */
}
```
